McNetik Run Privacy Policy

1. Responsible Party Identity

Ferran Perez Garcia

Email: ferranperez2001@gmail.com


2. Information Collected

The McNetik Run application collects the following data related to game progress:

  • "coins" (virtual currency).
  • "recordDistance" (maximum distance achieved).
  • "skins" (matrix referencing player obtained skins).

The collection and storage of this data is done through Firebase Firestore.

Anonymous authentication provided by Firebase Auth is used.


3. Purpose of Processing

The collected data is used exclusively to maintain game progress and allow users to unlock additional content within the application.

McNetik Run does not share data with third parties or display advertising.


4. Data Security

Standard security measures offered by Firebase are implemented to protect access to the database.

Since no sensitive personal data is collected, the basic security measures provided by Firebase are used.


5. User Rights

Since the collected data is intrinsically linked to game progress, its modification or deletion would affect the user experience. Users can influence this data by interacting with the game.


6. Children's Privacy

McNetik Run is suitable for all ages, but is not specifically aimed at children under 13. Therefore, the Children's Online Privacy Protection Act (COPPA) does not apply.


7. Changes to the Privacy Policy

Modifications to this privacy policy will be communicated through the developer's website (https://ferranperezgarcia.github.io/portfolio/McNetikRun.html), once it is available.


8. Additional Information

McNetik Run uses the Firebase Firestore and Firebase Auth SDKs.

Regulatory Compliance:

  • General Data Protection Regulation (GDPR): McNetik Run complies with the principles of the GDPR, although it does not collect sensitive personal data, it is noted that online identifiers are also subject to this regulation.
  • Google Play Policies: McNetik Run adheres to the privacy policies of the Google Play Store.
