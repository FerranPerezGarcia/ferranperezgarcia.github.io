McNetik Run Privacy Policy

1. Responsible Party Identity

Ferran Perez Garcia

Email: ferranperez2001@gmail.com


2. Information Collected

The McNetik Run application collects the following data related to game progress:

  • "coins" (virtual currency).
  • "recordDistance" (maximum distance achieved).
  • "skins" (matrix referencing player obtained skins).

The collection and storage of this data is done through Firebase Firestore.

Anonymous authentication provided by Firebase Auth is used.


3. Purpose of Processing

The collected data is used exclusively to maintain game progress and allow users to unlock additional content within the application.

McNetik Run does not share data with third parties or display advertising.


4. Data Security

Standard security measures offered by Firebase are implemented to protect access to the database.

Since no sensitive personal data is collected, the basic security measures provided by Firebase are used.


5. User Rights

Since the collected data is intrinsically linked to game progress, its modification or deletion would affect the user experience. Users can influence this data by interacting with the game.


6. Children's Privacy

McNetik Run is suitable for all ages, but is not specifically aimed at children under 13. Therefore, the Children's Online Privacy Protection Act (COPPA) does not apply.


7. Changes to the Privacy Policy

Modifications to this privacy policy will be communicated through the developer's website (https://ferranperezgarcia.github.io/portfolio/McNetikRun.html), once it is available.


8. Additional Information

McNetik Run uses the Firebase Firestore and Firebase Auth SDKs.

Regulatory Compliance:

  • General Data Protection Regulation (GDPR): McNetik Run complies with the principles of the GDPR, although it does not collect sensitive personal data, it is noted that online identifiers are also subject to this regulation.
  • Google Play Policies: McNetik Run adheres to the privacy policies of the Google Play Store.

    Política de Privacidad de McNetik Run

    1. Identidad del Responsable

    Ferran Perez Garcia

    Correo electrónico: ferranperez2001@gmail.com


    2. Información Recopilada

    La aplicación McNetik Run recopila los siguientes datos relacionados con el progreso del juego:

    • "coins" (monedas virtuales).
    • "recordDistance" (distancia máxima alcanzada).
    • "skins" (matriz que hace referencia a las apariencias obtenidas del jugador).

    La recopilación y el almacenamiento de estos datos se realizan mediante Firebase Firestore.

    Se utiliza la autenticación anónima proporcionada por Firebase Auth.


    3. Finalidad del Tratamiento

    Los datos recopilados se utilizan exclusivamente para mantener el progreso del juego y permitir a los usuarios desbloquear contenido adicional dentro de la aplicación.

    McNetik Run no comparte datos con terceros ni muestra publicidad.


    4. Seguridad de los Datos

    Se implementan las medidas de seguridad estándar ofrecidas por Firebase para proteger el acceso a la base de datos.

    Dado que no se recopilan datos personales sensibles, se utilizan las medidas de seguridad básicas proporcionadas por firebase.


    5. Derechos del Usuario

    Dado que los datos recopilados están intrínsecamente ligados al progreso del juego, su modificación o eliminación afectaría la experiencia del usuario. Los usuarios pueden influir en estos datos mediante la interacción con el juego.


    6. Privacidad Infantil

    McNetik Run es apta para todas las edades, pero no está dirigida específicamente a niños menores de 13 años. Por lo tanto, no se aplica la Ley de Protección de la Privacidad Infantil en Línea (COPPA).


    7. Cambios en la Política de Privacidad

    Las modificaciones a esta política de privacidad se comunicarán a través del sitio web del desarrollador (www.insertepagianweb.com), una vez esté disponible.


    8. Información Adicional

    McNetik Run utiliza los SDK de Firebase Firestore y Firebase Auth.

    Cumplimiento normativo:

    • Reglamento General de Protección de Datos (RGPD): McNetik Run cumple con los principios del RGPD, aunque no recopila datos personales sensibles, se tiene en cuenta que los identificadores en línea también están sujetos a esta normativa.
    • Políticas de Google Play: McNetik Run se adhiere a las políticas de privacidad de Google Play Store.
